# NLP Sentiment Analysis - Master's Thesis

This repository contains the complete research, implementation, and evaluation framework for my Master's Thesis in **Information Engineering & Computer Science**. The project focuses on exploring and benchmarking diverse deep learning architectures to optimize natural language processing (NLP) workflows for sentiment analysis tasks.

## 📋 Project Overview
The objective of this thesis was to evaluate model performance, training efficiency, and prediction accuracy across different generations of neural network architectures. By structuring end-to-end machine learning pipelines, the framework handles everything from raw text preprocessing to deep model evaluation.

## 🧠 Architectures Evaluated
The project implements, tests, and compares three primary deep learning structural methodologies:
* **Convolutional Neural Networks (CNNs):** Utilized for capturing local feature patterns and spatial keyphrase relationships in text.
* **Recurrent Neural Networks (RNNs / LSTMs):** Implemented to capture sequential and long-term dependencies within textual data.
* **Transformer Models:** Leveraged for state-of-the-art context awareness and attention-based semantic understanding.

## 🛠️ Technical Stack & Toolkit
* **Language:** Python
* **Core Frameworks:** Keras (for CNN & RNN implementations) & Hugging Face (for Transformer models)
* **Data Processing:** Pandas, NumPy, Scikit-learn
* **Workflows:** End-to-end ML pipelines (Preprocessing, Tokenization, Training, Cross-Validation, Evaluation)

## 🚀 Key Features Implemented
* **Robust Preprocessing Pipeline:** Automated text cleaning, tokenization, padding, and embedding mappings optimized for deep architectures.
* **End-to-End ML Pipelines:** Modular scripts designed for scalable training, checkpointing, and testing to ensure analysis readiness and full reproducibility.
* **Performance Benchmarking:** Detailed metric evaluation tracking and comparing training efficiency, prediction accuracy, and loss convergence profiles across all models.

## 📊 Evaluation Metrics
Models were rigorously benchmarked based on:
1. **Prediction Accuracy** (Precision, Recall, F1-score)
2. **Training Efficiency** (Compute time per epoch, resource scalability)
